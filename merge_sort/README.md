# Merge Sort
### Input
- **x** not sorted list
### processes
- ....
- merge 2 sorted lists:
    - i, j = 0 as each tracking the elements in the 2 arrays (a,b)
    - if a[i] < b[j] then 
        - take a[i] and put it the final array
        - increment i by 1
    - else
        - put a[j] in the sorted array
        - increment j by 1
### Output
- **y** as sorted list
